# syntiq-mt5 docs

Use these pages to get productive quickly:

- Start with **Quickstart**
- Learn **Result[T]** and **Lifecycle**
- Use **Tasks** for copy-paste workflows

## Available Tasks

- **Get positions** - Retrieve open positions
- **Get candles** - Fetch historical price data
- **Get account info** - Check balance, equity, and margin
- **Get symbols** - List and query tradeable instruments
- **Get ticks** - Retrieve tick-level price data
- **Get history** - Access historical orders and deals
- **Place orders** - Validate and execute trades
- **Market book** - Subscribe to Level 2 order book data
- **Handle errors** - Work with Result[T] error handling
